## 简单按钮
- 体积小
- 点击效果波纹效果

## API
| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | 类型，现在就两种，primary和simple | string | primary |
| style | 样式 | object | 无 |
| onChange | 点击事件 | Function | 无 |

按钮默认宽50px，高28px，如有需要自己传style调整
